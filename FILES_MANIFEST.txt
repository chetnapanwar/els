CORE APPLICATION FILES
================================================================================

1. main.py
   - FastAPI application entry point
   - Uvicorn server configuration
   - Router inclusion
   - Root endpoint

2. app/__init__.py
   - Package initialization

3. app/models/__init__.py
   - Models package initialization

4. app/models/user.py
   - UserCreateRequest (Pydantic model)
     * username: 1-50 characters
     * password: minimum 6 characters  
     * email: valid email format
   - UserCreateResponse (Pydantic model)
     * id: user ID
     * username: user's username
     * email: user's email
     * message: success message

5. app/services/__init__.py
   - Services package initialization

6. app/services/user_service.py
   - UserService class with:
     * create_user() - creates and validates user
     * get_all_users() - retrieves all users
     * _users_db - in-memory storage dictionary
     * Duplicate username prevention

7. app/routes/__init__.py
   - Routes package initialization

8. app/routes/user_routes.py
   - POST /users - Create user endpoint (201)
   - GET /users - Get all users endpoint (200)
   - Error handling with HTTPException
   - Status code management
